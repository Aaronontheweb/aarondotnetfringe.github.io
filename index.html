WHAT IS .NET FRINGE?

.NET Fringe is a new conference focused on .NET developers who are breaking the rules and pushing the boundaries. The .NET platform is more interesting today than it ever has been! The old Microsoft that was at odds with Open Source has been replaced with one that is actively contributing to OSS and hosts a huge number of important projects on GitHub! The .NET OSS community has been collaborating and creating new, really exciting, interesting, and innovative projects which are changing the landscape. They are defining the future!

Come be a part of this fantastic gathering of open source .NET developers!!!

A CONVERSATION

Have you heard there's this new *different* .NET conf call Fringe?
What? Isn't .NET dead?
I mean I remember I was excited about this ALT.NET thing
a while ago, but not sure what happened.
Node is the cool kid nowadays.
Dude, a lot has changed since the last time you checked!

Did you know there are over 200,000 C# projects on Github?

Did you hear about the NuGet package manager which has over 30,000 projects?

Did you know Microsoft killed their JSON library and is now shipping JSON.NET?

Did you know Microsoft is building a new nextgen node-like .NET stack
which is completely OSS on Github (As in taking PRs) and works on Linux and Mac?


Woah! I guess you are right. Things really have changed!

INFO

Venue

This year's conference will be held at the beautiful historic Bossanova Ballroom located at 722 East Burnside, Portland, Oregon. The Bossanova Ballroom was built in 1909, and is one of the 9 remaining arcade buildings in Portland Oregon.

Bossanova Ballroom is ADA accessible, but there is not an elevator. If you are, or have someone that is, in a wheelchair please call 503-206-7630 for our wheelchair accessibility options prior to event.

Hotels

If you're coming from out of town, consider staying at the stylish Jupiter Hotel or the more budget friendly Eastside Lodge which are both within a block of the venue. If those are already booked, consider staying in one of the many hotels in the near-by Lloyd District or Downtown Portland.
